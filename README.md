# arXiv CS Paper Fetcher



A script that automatically collects titles and abstracts of the latest Computer Science (CS) papers from arXiv and saves them to a text file.



## Features

- Search for the latest CS papers using the arXiv API

- Extract titles, links, and abstracts

- Save results to a clean, readable `.txt` file



## Installation \& Usage



1. Clone the repository

```
bash

git clone \[https://github.com/YOUR\_USERNAME/arxiv\_cs\_collector.git](https://github.com/YOUR\_USERNAME/arxiv\_cs\_collector.git)

cd arxiv\_cs\_collector
```

2. Run main.py

```
python main.py --YOUR_CUSTOM_OPTIONS
```
